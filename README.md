# AviUtl RAMプレビュー

メモリに動画を読み込み、プレビュー再生を行うプラグインです。  
拡張編集等で、編集の重さに左右されない動画の再生確認ができます。  

## 導入
AviUtl本体のあるディレクトリに plugins フォルダを作成し、当プラグインを入れます。  
正しく導入できれば、プラグインフィルタ情報にプラグイン名が追加されています。  

## 使い方
1. 本体のメニューより 表示→RAMプレビューの表示 を選択し、RAMプレビューのウィンドウを表示します。  
2. 動画を読み込み・作成後、RAMプレビューウィンドウのメニューより 再生→読み込み  
   またはウィンドウ右下のうち一番左の読み込みボタンを押します。
3. 読み込み完了後、 再生→再生 または右下中央の再生ボタンを押します。
